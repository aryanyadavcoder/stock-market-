#  Trade Aryan - Stock Trading & Analysis App

## About Project
Trade Aryan is a Python-based GUI stock trading and analysis application built using Tkinter.

It allows users to analyze stocks, compare multiple stocks, and perform demo trading with advanced features like auto-trading and portfolio tracking.


##  Features

###  Stock Analysis
- View stock price charts
- Daily return %
- Volume & histogram analysis
- Statistics table (Mean, Median, Min, Max, Std Dev)

###  Stock Comparison
- Compare 2 stocks
- Close price comparison
- Normalized performance
- Group Average (2-day / 7-day)

###  Chart Types
- Line Chart
- Bar Chart
- Scatter Plot

### Demo Trading System
- Create users
- Deposit cash
- Buy/Sell stocks
- Portfolio tracking
- Transaction history

###  Auto Trading (Advanced)
- Set target price
- Auto Buy/Sell execution
- Real-time price checking

###  Extra Features
- Market snapshot saving
- Live stock price display
- Multi-threading for smooth UI



##  Technologies Used

- Python
- Tkinter (GUI)
- yfinance (Stock Data API)
- Matplotlib (Data Visualization)
- Pandas (Data Processing)
- Multithreading (threading module)

##  Usage

### Analyze Stock
- Enter stock name (e.g., RELIANCE.NS)
- Click "Analyze Share"
- View charts and statistics

### Compare Stocks
- Enter two stock tickers
- Click "Compare Shares"

### Demo Trading
- Create user
- Deposit money
- Buy/Sell stocks

### Auto Trading
- Set target price
- Enable Auto-Trade
